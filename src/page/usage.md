# Usage

<br />

Import components directly and use it.


## Composition API

Import with composition API

```vue
<script setup>
import { WButton } from "vue-windi";
</script>

```

## Options API


import with options API 

```vue
<script>
import { WButton } from "vue-windi";

export default {
    components: {
        WButton
    }
}
</script>

```
