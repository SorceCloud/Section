# Usage

<br />

Import components directly and use it.


## Composition API

Import with composition API

```vue
<script setup lang="ts">
import btn from "section";
</script>

```

## Options API


import with options API 

```vue
<script lang="ts">
import btn from "sectionui";

export default {
    components: {
        btn
    }
}
</script>

```
