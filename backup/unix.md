## Solution

### List your shells

```sh
cat /etc/shells
```

### Find your current shell

```sh
echo $SHELL
```

### Change your default shell

```sh
chsh -s /path/to/shell
```

Examples:
```sh
chsh -s /bin/sh
chsh -s /bin/bash
chsh -s /bin/zsh
```
